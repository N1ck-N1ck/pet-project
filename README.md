You need to install Node.js version 22.12.0 and create a database using the settings from db.js.
The database should contain a table named scooters with the following columns:
scooter_name
image
about
id
price
The id column should be of type serial and set as the primary key.
The image column should store the path to scooter images located in the image directory:
imgAovo.png, imgMidway.png.
